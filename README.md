# About
Simple plain JavaScript project using [Vite.js](https://vitejs.dev/guide/) template and [vite-plugin-pwa](https://www.npmjs.com/package/vite-plugin-pwa).

# How to Run

## Install Dependencies
`pnpm install`

## Development Mode

`npm run dev`

## Production Mode

`npm run preview`

## Build Production
`npm run build`
